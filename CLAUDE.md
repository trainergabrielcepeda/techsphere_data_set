# CLAUDE.md

Guidance for Claude Code when working in this repository.

## Workflow rules

- **Plan before every change.** For each change to the project, first write a plan
  in the format `##-name-of-plan` (a descriptive, kebab-case name).
- **Write the plan to `Plans/`.** The plan must be written as a file under `Plans/`
  (e.g. `Plans/02-name-of-plan.md`) so the user can review it there, not just
  presented inline in chat.
- **Wait for approval.** After presenting a plan, MANDATORY: stop and wait for the
  user's explicit approval before making any change. Do not start implementing until
  the user approves the plan.
